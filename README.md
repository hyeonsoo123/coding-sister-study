# 임현수 · 백엔드 개발자 포트폴리오

2년차 백엔드 개발자 임현수의 포트폴리오 사이트입니다.
정적 웹(HTML/CSS/JS)으로 만들었으며, 데모로 **캘린더 기반 TODO 앱**과 **테트리스 미니게임 테마**를 포함합니다.

## 📄 페이지 구성

- **`index.html`** — 포트폴리오 메인 (소개 / 주요 프로젝트 / 연락처)
- **`todo.html`** — 캘린더 기반 TODO 관리 데모 앱
- **테마 전환** — 기본 테마 ↔ 테트리스(네온 아케이드) 테마. 테트리스 테마에서는 플레이 가능한 미니게임이 나타납니다.

## ✨ TODO 앱 기능

- **📅 캘린더 뷰**: 월별 캘린더에서 날짜 선택, 작업 있는 날짜 강조
- **✅ 작업 관리**: 제목·설명·우선순위(높음/중간/낮음)와 함께 등록
- **📊 필터링**: 전체 / 진행중 / 완료
- **⏱️ 기록**: 생성·완료 시각 자동 저장
- **💾 저장**: 브라우저 localStorage (서버·DB 없이 동작)
- **📱 반응형**: 모바일(탭 전환) / 데스크톱(캘린더+목록 3열) 레이아웃

## 🎮 테트리스 미니게임

- Canvas API로 구현 (7종 블록, 회전·충돌·라인 제거)
- 키보드(방향키/스페이스/P) + 모바일 터치 버튼 조작 지원

## 📁 프로젝트 구조

```
coding-sister-todo/
├── public/                 # 배포 대상 정적 파일
│   ├── index.html          # 포트폴리오 메인
│   ├── todo.html           # TODO 앱
│   ├── css/
│   │   ├── responsive.css  # 반응형 스타일
│   │   └── themes.css      # 테트리스 테마 스타일
│   └── js/
│       ├── app.js          # 앱 초기화·탭 전환
│       ├── calendar.js     # 캘린더 렌더링
│       ├── todo.js         # TODO 로직
│       ├── tetris.js       # 테트리스 게임
│       └── themes.js       # 테마 전환
├── src/css/input.css       # Tailwind 입력 파일
├── package.json
└── README.md
```

## 🚀 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:8081` 접속.

## 💻 기술 스택

- **HTML5 / CSS3** (Tailwind CSS + 커스텀 반응형 스타일)
- **JavaScript (ES6+)** — 캘린더·TODO·테트리스 로직, Canvas API
- **LocalStorage** — 클라이언트 데이터 저장
- **live-server** — 개발 서버

## 📝 참고

- 모든 데이터는 브라우저 localStorage에만 저장되며, 캐시를 지우면 사라집니다.
- 백엔드·DB 없이 동작하는 정적 사이트입니다.
